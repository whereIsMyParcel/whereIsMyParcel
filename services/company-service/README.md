## Company-service

해당 서비스는 업체를 등록하고 업체 직원을 관리하는 서비스를 담당합니다

### company 

- 업체 등록 : 허브 매니저 혹은 마스터만 가능합니다
  - feign 통신으로 request에 사업자번호와 유저의 사업자 번호가 일치한 유저의 아이디를 받아옵니다
  - 해당 아이디를 기반으로 초기 멤버를 지정합니다
- 업체 삭제 : 허브 매니저 혹은 마스터만 가능합니다
  - feign 통신으로 해당 업체 아이디를 가진 모든 유저의 업체 아이디르 지워달라고 요청합니다
    - (유저를 지워도 될지는 모르지만 그래도 될것 같습니다)

---

### CompanyMember
- 멤버 등록 : 업체 매니저만 가능합니다
  - feign 통신으로 등록할 유저의 아이디를 받아옵니다
    - 해당 유저이 다른 허브아이디나 컴퍼니 아이디가 있는지 검증이 필요할것 같습니다
    - 만약 있다면 에러를 반환해주세요
- 멤버 삭제 : 허브 매니저 혹은 마스터만 가능합니다
  - feign 통신으로 유저에게 해당 유저의 업체 아이디를 지워달라고 요청을 보냅니다
  - (유저를 지워도 될지는 모르지만 그래도 될것 같습니다)
